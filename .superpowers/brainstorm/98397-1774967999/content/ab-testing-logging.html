A/B-testlogik

  • När A/B är på: slumpar bundle vid varje ny session (app-start). Visas diskret i UI så du vet vilken du testar.
  • När A/B är av: använder vald bundle från settings.
  • Fallback: om vald provider saknar API-nyckel → Open automatiskt.
  • Exporterbar data: alla loggar kan exporteras som JSON så du kan dela med mig.
