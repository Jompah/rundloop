  <action>
    **Create `src/lib/crash-recovery.ts`** with:

    1. **Module state**:
       ```typescript
       let snapshotPointCount = 0;
       let snapshotTimer: ReturnType<typeof setInterval> | null = null;
       ```

    2. **`startSnapshotSchedule(getRunState: () => ActiveRunSnapshot): void`**:
       - Reset `snapshotPointCount = 0`
       - Clear any existing timer
       - Set `snapshotTimer = setInterval(() => { saveSnapshot(getRunState()) }, 10_000)` (every 10 seconds)

    3. **`stopSnapshotSchedule(): void`**:
       - If `snapshotTimer`, call `clearInterval(snapshotTimer)`, set to null
       - Reset `snapshotPointCount = 0`

    4. **`onPointAccepted(getRunState: () => ActiveRunSnapshot): void`**:
       - Increment `snapshotPointCount`
       - If `snapshotPointCount >= 30`: reset to 0, call `saveSnapshot(getRunState())`

    5. **Private `saveSnapshot(snapshot: ActiveRunSnapshot): void`**:
       - Call `dbPut('runs', snapshot)` — the same ID overwrites previous snapshot
       - Wrap in try/catch, log error but don't throw (snapshot failure must not crash run)

    6. **`findIncompleteRun(): Promise<ActiveRunSnapshot | null>`**:
       - Call `dbGetAll<ActiveRunSnapshot | CompletedRun>('runs')`
       - Find first record where `startTime` exists AND `'endTime' in record === false` (ActiveRunSnapshot has no endTime field)
       - Return it, or null if none found
       - Distinguish by checking: `if (!('endTime' in run))` — ActiveRunSnapshot lacks endTime, CompletedRun has it

    7. **`clearIncompleteRun(id: string): Promise<void>`**:
       - Call `dbDelete('runs', id)` — removes the incomplete run record (user chose "Discard")
  </action>
