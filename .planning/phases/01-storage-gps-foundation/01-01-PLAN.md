<objective>
Create the IndexedDB persistence layer that replaces localStorage for all app data (runs, routes, settings), migrate existing localStorage data on first load, and request storage persistence to prevent iOS eviction.

Purpose: All subsequent phases depend on reliable persistent storage. localStorage has a 5MB limit, is synchronous, and gets evicted by iOS after 7 days of inactivity.
Output: `src/lib/db.ts` with IndexedDB CRUD, updated `src/types/index.ts` with Run types, updated `src/lib/storage.ts` delegating to IndexedDB.
</objective>

  <action>
    **Create `src/lib/db.ts`** with the following:

    1. **Singleton IndexedDB connection** using `indexedDB.open('rundloop', 1)`:
       - `onupgradeneeded`: Create 3 object stores:
         - `runs` with keyPath `id` and index `startTime` on field `startTime` (unique: false)
         - `routes` with keyPath `id`
         - `settings` with keyPath `key`
       - `onblocked`: `console.warn('IndexedDB upgrade blocked by another tab')`
       - `onsuccess`: Cache in `dbInstance` variable, add `db.onversionchange = () => { db.close(); dbInstance = null; }`
       - SSR guard: if `typeof window === 'undefined'`, reject with error

    2. **Generic CRUD helpers** (all return Promises):
       - `dbGet<T>(storeName: string, key: string | number): Promise<T | undefined>` — single record read
       - `dbPut<T>(storeName: string, value: T): Promise<void>` — upsert record
       - `dbDelete(storeName: string, key: string | number): Promise<void>` — delete record
       - `dbGetAll<T>(storeName: string): Promise<T[]>` — all records from store
       - `dbGetAllByIndex<T>(storeName: string, indexName: string, direction?: IDBCursorDirection): Promise<T[]>` — records via index cursor, default direction 'prev' (newest first)

    3. **Migration function** `migrateFromLocalStorage(): Promise<void>`:
       - Check `localStorage.getItem('rundloop_migrated_to_idb')` — if truthy, return early
       - Read `localStorage.getItem('rundloop_settings')`, parse JSON, write to settings store as `{ key: 'app', ...parsedSettings }`
       - Read `localStorage.getItem('rundloop_history')`, parse JSON as `SavedRoute[]`, write each to routes store via `dbPut`
       - Set `localStorage.setItem('rundloop_migrated_to_idb', 'true')`
       - Remove `localStorage.removeItem('rundloop_settings')` and `localStorage.removeItem('rundloop_history')`
       - Wrap each parse in try/catch — ignore corrupt data silently

    4. **Storage persistence** `requestPersistentStorage(): Promise<boolean>`:
       - SSR guard: return false if `typeof window === 'undefined'`
       - Check `navigator.storage?.persist` exists, return false if not
       - Call `navigator.storage.persisted()` — if already persisted, return true
       - Call `navigator.storage.persist()` and return result
       - Wrap in try/catch, return false on error

    5. **DB ready gate** `initDB(): Promise<void>`:
       - Calls `getDB()` to ensure connection is open
       - Calls `migrateFromLocalStorage()`
       - Calls `requestPersistentStorage()`
       - This is the single entry point consumers call before any DB operation
