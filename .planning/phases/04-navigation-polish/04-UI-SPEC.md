## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, gap-2 |
| md | 16px | Default element spacing, p-4 |
| lg | 24px | Section padding, p-6 |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Touch targets for map controls (re-center button, voice toggle) use 48px minimum hit area (12px padding on 24px icon) for gloved/sweaty finger tapping during runs.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 400 (regular) | 1.5 |
| Heading | 20px | 700 (bold) | 1.2 |
| Display | 48px | 700 (bold) | 1.1 |

Source: Existing codebase patterns. Display size (text-5xl / 48px) matches RunMetricsOverlay pace readout. Heading size (text-xl / 20px) matches SettingsView header. Label size (text-xs / 12px) matches metric labels ("Pace", "Distance"). Body size (text-sm / 14px) matches instruction text.

## Phase-Specific Visual Contracts

### Off-Route Banner (NAV-02)

| Property | Value |
|----------|-------|
| Position | Fixed top, below navigation instruction bar |
| Background | `bg-amber-500/20` with `backdrop-blur-sm` |
| Border | `border border-amber-500/50` top and bottom |
| Text color | `text-amber-200` for direction, `text-white` for "Off route" label |
| Icon | Inline SVG warning triangle, 20px, `text-amber-400` |
| Layout | Row: [warning-icon] [text-stack] |
| Text stack | Line 1: "Off route" (14px, semibold, white). Line 2: "Head [direction] to rejoin" (12px, regular, amber-200) |
| Animation | Slide down from navigation bar with 200ms ease-out. Fade out over 300ms when back on route. |
| Z-index | z-25 (above map, below navigation instruction bar z-20) |
| Haptic | `navigator.vibrate([200, 100, 200])` on first appearance per off-route event |
| Repeat behavior | Banner stays visible while off-route. Voice repeats every 30 seconds if still off-route. |
| Dismissal | Auto-dismisses when runner returns within 50m of route |

### Re-Center Button (NAV-01)

| Property | Value |
|----------|-------|
| Position | `absolute bottom-[220px] right-4` (above metrics overlay, aligned with existing center-on-user button position) |
| Size | 48px x 48px (touch target) |
| Background (auto-rotating) | `bg-gray-900/90` with white compass icon |
| Background (manual override) | `bg-white` with dark compass icon + green dot indicator |
| Border radius | `rounded-full` |
| Shadow | `shadow-lg` |
| Icon | Inline SVG compass/navigation arrow, 20px |
| Active state | `active:bg-gray-100` (same as existing center button) |
| Visibility | Only visible during active navigation when `isAutoRotating === false` |
| Behavior | Tap resets `isAutoRotating = true` and triggers immediate `easeTo` with current heading |

### Voice Style Selector in Settings (NAV-03, NAV-04)

| Property | Value |
|----------|-------|
| Position | Below existing "Voice Navigation" toggle in SettingsView |
| Visibility | Only visible when `voiceEnabled === true` |
| Layout | Vertical stack of 3 radio-style pills, full width |
| Pill height | 52px (comfortable touch target) |
| Pill background (inactive) | `bg-gray-800 text-gray-300` |
| Pill background (active) | `bg-green-500 text-white` |
| Pill border radius | `rounded-xl` (matches existing unit selector) |
| Pill content | Row: [style-name] [example-text]. Style name: 14px semibold. Example: 12px regular gray-400 |
| Style names | "Concise", "With Pace", "Motivational" |
| Example texts | "1 km completed", "1 km completed. Pace: 5:30/km", "Great work! 1 km done" |
| Transition | `transition-colors` (matches existing unit buttons) |
| Gap between pills | 8px (gap-2) |

### Map Navigation Mode (NAV-01)

| Property | Value |
|----------|-------|
| Pitch during navigation | 30 degrees (forward-looking perspective) |
| Zoom during navigation | 16 (street-level detail) |
| EaseTo duration | 500ms per GPS update |
| Bearing source | `FilteredPosition.heading` when speed > 1.0 m/s, else last calculated bearing from 2 points |
| Heading smoothing | Exponential moving average, alpha = 0.3 |
| User marker during navigation | Same blue dot (16px, #3b82f6, white border, blue shadow) -- unchanged |

## Interaction States

### Off-Route State Machine

| State | Visual | Audio | Haptic |
|-------|--------|-------|--------|
| On route | No banner | Normal milestone cues | None |
| Transition to off-route | Banner slides in (200ms) | "Off route. Head {dir} to rejoin." | vibrate([200, 100, 200]) |
| Sustained off-route (>30s) | Banner remains visible | Repeat voice announcement every 30s | None (only on initial) |
| Return to on-route | Banner fades out (300ms) | "Back on route." | None |

### Auto-Rotation State Machine

| State | Map Behavior | Re-center Button |
|-------|-------------|-----------------|
| Auto-rotating (default) | Follows heading, pitch 30, zoom 16 | Hidden |
| Manual override (user panned/zoomed) | Static, user-controlled | Visible (white bg, compass icon) |
| Re-centered (button tapped) | Snaps back to heading-following mode | Hides after snap |
