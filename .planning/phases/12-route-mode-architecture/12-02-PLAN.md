# Plan 12-02: Scenic Mode Toggle UI and Wiring

**Phase:** 12 - Route Mode Architecture
**Wave:** 2
**Depends on:** 12-01
**Files modified:** `src/components/RouteGenerator.tsx`, `src/app/page.tsx`
**Autonomous:** Yes
**Requirements addressed:** [ROUTE-01, ROUTE-04]

## Objective

Add a three-segment scenic mode toggle (Standard / Natur / Utforska) to the RouteGenerator UI. The toggle is disabled when algorithmic mode is selected. Wire the scenic mode preference through page.tsx to the AI prompt pipeline and persist it in IndexedDB.

## Tasks

### Task 1: Add scenic mode toggle UI to RouteGenerator

<read_first>
- src/components/RouteGenerator.tsx
- src/types/index.ts
</read_first>

<action>
1. Import `ScenicMode` from `@/types`

2. Add props to `RouteGeneratorProps`:
   ```typescript
   scenicMode?: ScenicMode;
   onScenicModeChange?: (mode: ScenicMode) => void;
   ```

3. Add prop destructuring in component function signature: `scenicMode, onScenicModeChange`

4. Add the scenic mode toggle UI between the city name div and the distance display div. The toggle should be:
   ```tsx
   {/* Scenic mode toggle - only for AI routes */}
   {routeMode === 'ai' && (
     <div className="flex gap-1 mb-4 bg-gray-800 rounded-xl p-1">
       {([
         { value: 'standard' as ScenicMode, label: 'Standard' },
         { value: 'nature' as ScenicMode, label: 'Natur' },
         { value: 'explore' as ScenicMode, label: 'Utforska' },
       ]).map(({ value, label }) => (
         <button
           key={value}
           onClick={() => onScenicModeChange?.(value)}
           className={`flex-1 py-2 rounded-lg text-sm font-medium transition-colors min-h-[44px] ${
             scenicMode === value
               ? 'bg-green-500 text-white'
               : 'text-gray-400 active:bg-gray-700'
           }`}
         >
           {label}
         </button>
       ))}
     </div>
   )}
   ```

5. Place this JSX block right after the city name div (`<div className="text-gray-400 text-sm mb-1">`) and before the distance display div (`<div className="text-center mb-4">`).
</action>

<acceptance_criteria>
- `RouteGeneratorProps` contains `scenicMode?: ScenicMode` and `onScenicModeChange?: (mode: ScenicMode) => void`
- Toggle renders three buttons with labels "Standard", "Natur", "Utforska"
- Toggle is wrapped in `{routeMode === 'ai' && (...)}`  so it only shows for AI mode
- Active button has `bg-green-500 text-white` class
- Each button has `min-h-[44px]` for touch targets
- Swedish labels used: "Standard", "Natur", "Utforska"
</acceptance_criteria>

### Task 2: Wire scenic mode state in page.tsx

<read_first>
- src/app/page.tsx
- src/types/index.ts
- src/lib/storage.ts
</read_first>

<action>
1. Import `ScenicMode` from `@/types` (add to existing import)

2. Add state for scenic mode after the `routeMode` state:
   ```typescript
   const [scenicMode, setScenicMode] = useState<ScenicMode>('standard');
   ```

3. Load persisted scenic mode from IndexedDB. Inside the existing `useEffect` that loads settings (or create a new one after component initialization):
   ```typescript
   useEffect(() => {
     getSettings().then((s) => {
       if (s.scenicMode) setScenicMode(s.scenicMode);
     });
   }, []);
   ```
   Note: If there is already a settings load effect, add the scenic mode loading there instead of creating a duplicate.

4. Create a handler to update scenic mode and persist it:
   ```typescript
   const handleScenicModeChange = useCallback(async (mode: ScenicMode) => {
     setScenicMode(mode);
     const settings = await getSettings();
     await saveSettings({ ...settings, scenicMode: mode });
   }, []);
   ```

5. Import `saveSettings` from `@/lib/storage` (add to existing import)

6. Pass scenic mode props to `<RouteGenerator>`:
   ```tsx
   scenicMode={scenicMode}
   onScenicModeChange={handleScenicModeChange}
   ```

7. Pass scenic mode to `generateRouteWaypoints` call in `handleGenerate`. Change:
   ```typescript
   initialWaypoints = await generateRouteWaypoints({ lat: startLat, lng: startLng, distanceKm: distance, cityName, settings });
   ```
   To:
   ```typescript
   initialWaypoints = await generateRouteWaypoints({ lat: startLat, lng: startLng, distanceKm: distance, cityName, settings, scenicMode });
   ```

8. Add `scenicMode` to the `handleGenerate` useCallback dependency array.
</action>

<acceptance_criteria>
- `page.tsx` contains `const [scenicMode, setScenicMode] = useState<ScenicMode>('standard')`
- `page.tsx` contains `handleScenicModeChange` callback that calls `saveSettings`
- `RouteGenerator` component receives `scenicMode={scenicMode}` and `onScenicModeChange={handleScenicModeChange}` props
- `generateRouteWaypoints` call includes `scenicMode` in the request object
- `saveSettings` is imported from `@/lib/storage`
- `handleGenerate` dependency array includes `scenicMode`
</acceptance_criteria>
