## Interaction Contracts

### Card Tap (History)
- Tap anywhere on RunHistoryCard opens RunDetailOverlay
- Provide active:bg-gray-700 feedback on press

### Card Actions (Saved Routes)
- Tap card name area enters inline edit mode
- Tap "Run" button loads route onto map and transitions to navigation
- Long-press or swipe: not implemented (use detail view for delete)

### Delete Flow
- In RunDetailOverlay, tap "Delete Run" button
- DeleteRunDialog appears with backdrop
- "Keep It" dismisses dialog, "Delete" removes from IndexedDB and returns to history list
- Saved route deletion follows same pattern from a route detail or swipe action

### Tab Navigation
- Tap tab to switch view (generate / history / routes)
- No swipe-between-tabs gesture (keep simple)
- Active tab shows green-400, inactive shows gray-500
- Tab bar hidden during navigate, summary, settings views

### Inline Name Editing (Saved Routes)
- Tap route name text to enter edit mode
- Text replaced by focused input field
- Blur or Enter key saves the new name to IndexedDB
- Escape key cancels edit and reverts to previous name
- Empty name reverts to auto-generated name

### List Scrolling
- Vertical scroll within tab content area
- Content area: from top safe area to tab bar top edge
- No pull-to-refresh (data loads on view mount and after mutations)
