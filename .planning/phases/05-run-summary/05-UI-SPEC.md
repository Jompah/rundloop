| Text secondary | #9ca3af | text-gray-400 | Stat labels, body text, calorie note |

Accent reserved for: "Save Run" CTA button background, highlighted stat values where the runner performed well (distance, pace). Never used on destructive or neutral elements.

Planned route line: `#4ade80` (green-400) at 3px width.
Actual GPS trace line: `#22d3ee` (cyan-400) at 3px width.
This gives clear visual distinction between planned and actual on the summary map.

## Component Inventory

### RunSummaryView (new)

Full-screen view replacing the navigation view after run completion.

**Layout (top to bottom):**
1. **Summary map** — 50% viewport height, renders planned route (green) + actual trace (cyan), auto-fitted to bounds with 40px padding
2. **Stats card** — bg-gray-800 rounded-2xl, mx-4, p-6. 2x2 grid of stat cells
3. **Calorie note** — text-sm text-gray-400, centered below stats card. Only shown when weight is default (70kg)
4. **Action buttons** — Two buttons side by side in flex row, gap-3, mx-4, pb safe-area

**Stats card grid (2x2):**
| Cell | Label | Value format | Color |
|------|-------|-------------|-------|
| Top-left | "Distance" | formatMetricDistance + unit label | text-white |
| Top-right | "Time" | formatElapsed (HH:MM:SS) | text-white |
| Bottom-left | "Avg Pace" | formatPace + /km or /mi | text-green-400 |
| Bottom-right | "Calories" | rounded integer + "kcal" | text-white |

**Fade-in animation:** Component mounts with `opacity-0`, transitions to `opacity-100` via `transition-opacity duration-500 ease-out` triggered by useEffect state toggle on mount.

### DiscardConfirmDialog (new)

Modal dialog following EndRunDialog pattern exactly.

**Structure:**
- Fixed overlay: `fixed inset-0 z-50 flex items-center justify-center bg-black/60`
- Card: `bg-gray-900 rounded-2xl p-6 mx-4 max-w-sm w-full`
- Heading: "Discard Run?" — `text-white text-xl font-bold`
- Body: "This run will be permanently deleted." — `text-gray-400 mt-2`
- Buttons: `flex gap-3 mt-6`
  - Cancel: "Keep It" — `flex-1 py-3 rounded-xl font-semibold bg-gray-800 text-white active:bg-gray-700`
  - Confirm: "Discard" — `flex-1 py-3 rounded-xl font-semibold bg-red-500 text-white active:bg-red-600`

### Settings Weight Input (modification to SettingsView)

- New section in SettingsView below existing fields
- Label: "Body Weight" — same label style as other settings
- Numeric input field: `bg-gray-800 text-white rounded-lg px-4 py-3 w-full`
- Unit suffix displayed inline: "kg" or "lbs" based on units setting
- Placeholder: "70" (or "154" for imperial)
- Internal storage always in kg; display converts when units === 'miles'

## Interaction States

### Summary View Lifecycle
1. **Entry:** User confirms "End Run" in EndRunDialog. CompletedRun is captured. View switches to summary.
2. **Visible:** Fade-in over 500ms. Map renders with fit-to-bounds. Stats populate immediately.
3. **Save:** User taps "Save Run". Run remains in IndexedDB. View returns to map (route generator). Route state cleared.
4. **Discard:** User taps "Discard". DiscardConfirmDialog appears.
   - "Keep It" dismisses dialog, returns to summary.
   - "Discard" deletes run from IndexedDB via `dbDelete('runs', id)`. View returns to map. Route state cleared.

### Button States
| Button | Default | Active/Pressed | Disabled |
|--------|---------|----------------|----------|
| Save Run | bg-green-500 text-white rounded-xl py-3 font-semibold | bg-green-600 (active:bg-green-600) | N/A |
| Discard | bg-gray-800 text-white rounded-xl py-3 font-semibold | bg-gray-700 (active:bg-gray-700) | N/A |

Note: "Save Run" uses green-500 fill (not outline) as the primary action. "Discard" uses gray-800 as the lower-emphasis action. This inverts the EndRunDialog pattern intentionally: saving is the happy path here, discarding is secondary.

### Map Behavior
- Non-interactive: scroll/zoom/pan disabled on summary map (static display)
- Auto-fit bounds to encompass both planned route and actual trace with 40px padding
- Two polyline layers: planned route (green-400, 3px, opacity 0.6) underneath, actual trace (cyan-400, 3px, opacity 1.0) on top
- Start marker: green circle at trace[0]
- End marker: green circle with checkmark at trace[last]
