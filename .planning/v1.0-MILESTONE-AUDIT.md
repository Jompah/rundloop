## Integration Gaps (Non-Blocking Tech Debt)

### 1. Route polyline not saved with CompletedRun (HIGH)
- **Affects:** HIST-03, VIZ-01, VIZ-03
- **Fix:** Augment CompletedRun with `routePolyline` in page.tsx before IndexedDB save; pass `route?.id` to `startRun()`

### 2. Units setting not propagated to History/Detail views (MEDIUM)
- **Affects:** METR-03, HIST-01
- **Fix:** Add `getSettings()` load in RunHistoryView and RunDetailOverlay; pass `units` to formatters

### 3. Calorie formula hardcoded in RunDetailOverlay (MEDIUM)
- **Affects:** SUMM-04
- **Fix:** Replace inline formula with `estimateCalories()` + `settings.bodyWeightKg`

### 4. PWA icons missing (HIGH)
- **Affects:** PWA-01
- **Fix:** Generate and add `icon-192.png` and `icon-512.png` to `public/`

### 5. RouteGenerator ignores defaultDistance setting (LOW)
- **Affects:** UI-05
- **Fix:** Seed `distance` state from `getSettings().defaultDistance` on mount

### 6. Dead server API route (INFO)
- **Affects:** None (app works correctly via browser-direct Anthropic calls)
- **Fix:** Remove `/api/generate-route/` directory or wire it up for server-side AI calls
